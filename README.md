# React ラーニングプロジェクト

このリポジトリは、React を学習する過程で作成した練習プロジェクトをまとめたものです。

## プロジェクト一覧

### 1. React-Practice-Project-01---EatSplit
- **学習内容**：コンポーネント間通信、状態のリフトアップ、フォーム処理

### 2. React-Practice-Project-02---UsePopcorn
- **学習内容**：カスタムフック、useEffect、非同期データ取得

### 3. Student-Management-System
- **学習内容**：CRUD 操作、リストレンダリング、状態管理

### 4. Fire-Calculator-v1.0
- **学習内容**：フォーム処理、数値計算、条件付きレンダリング

### 5. weather-app
- **学習内容**：API リクエスト、useEffect、条件付きレンダリング

### 6. weather-app-ts
- **学習内容**：TypeScript の型定義、インターフェース、ジェネリクス

## 使用技術
- React、JavaScript、TypeScript
